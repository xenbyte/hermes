## Hermes

Hermes scrapes real estate websites for new rental listings in The Netherlands, broadcasts the results via Telegram, and provides AI-powered analysis of listings to help you find the perfect home.

> This project is a fork of [wtfloris/hestia](https://github.com/wtfloris/hestia), customized for additional needs and extended with AI analysis capabilities for scraped listings.

### Features

- **Real-time scraping** of Dutch rental websites every 5 minutes
- **Telegram notifications** for new listings matching your filters (city, price, size, agencies)
- **AI enrichment** using Claude to analyze listings against your personal profile
- **Web dashboard** for managing filters and browsing results
- **Multi-language support** (English and Dutch)
